# ระบบคำนวณค่าจัดส่งสินค้า (Shipping Cost Calculator)

ระบบคำนวณค่าจัดส่งสินค้าอัตโนมัติ พัฒนาด้วยภาษา Java

## คุณสมบัติ (Features)

- คำนวณค่าจัดส่งตามน้ำหนักสินค้า
- รองรับ 2 โซนปลายทาง: กรุงเทพฯ/ปริมณฑล และต่างจังหวัด
- รองรับบริการจัดส่งด่วน (Express)
- รองรับส่วนลดสำหรับสมาชิก VIP

## อัตราค่าจัดส่ง (Shipping Rates)

### กรุงเทพฯ และปริมณฑล

| ช่วงน้ำหนัก     | ราคา     |
|-----------------|----------|
| 0 - 1 กก.       | 40 บาท   |
| มากกว่า 1 - 3 กก. | 60 บาท   |
| มากกว่า 3 - 5 กก. | 80 บาท   |
| มากกว่า 5 กก.    | 100 บาท  |

### ต่างจังหวัด

| ช่วงน้ำหนัก     | ราคา     |
|-----------------|----------|
| 0 - 1 กก.       | 60 บาท   |
| มากกว่า 1 - 3 กก. | 90 บาท   |
| มากกว่า 3 - 5 กก. | 120 บาท  |
| มากกว่า 5 กก.    | 150 บาท  |

### บริการเสริม

| ตัวเลือก        | รายละเอียด              |
|-----------------|-------------------------|
| บริการด่วน      | +30 บาท                 |
| สมาชิก VIP      | ลด 20% จากค่าจัดส่งรวม   |

## ความต้องการของระบบ (Requirements)

- Java JDK 8 หรือสูงกว่า

## วิธีการใช้งาน (How to Use)

### คอมไพล์โปรแกรม (Compile)

```bash
javac -encoding UTF-8 ShippingCalculator.java
```

### รันโปรแกรม (Run)

```bash
java ShippingCalculator
```

### ตัวอย่างการใช้งาน (Example)

```
==========================================
    ระบบคำนวณค่าจัดส่งสินค้า
    Shipping Cost Calculator
==========================================

กรุณาระบุน้ำหนักสินค้า (กิโลกรัม): 2.5

เลือกปลายทาง:
1. กรุงเทพฯ และปริมณฑล
2. ต่างจังหวัด
กรุณาเลือก (1 หรือ 2): 1

ต้องการบริการด่วน (Express)? (1 = ใช่, 0 = ไม่): 1
เป็นสมาชิก VIP หรือไม่? (1 = ใช่, 0 = ไม่): 1

==========================================
           รายละเอียดค่าจัดส่ง
==========================================
น้ำหนักสินค้า: 2.50 กิโลกรัม
ปลายทาง: กรุงเทพฯ และปริมณฑล
บริการด่วน: ใช่ (+30 บาท)
สมาชิก VIP: ใช่ (ลด 20%)
------------------------------------------
ค่าจัดส่งรวม: 72.00 บาท
==========================================
```

## สูตรการคำนวณ (Calculation Formula)

```
ค่าจัดส่งรวม = (ค่าจัดส่งพื้นฐาน + ค่าบริการด่วน) × (1 - ส่วนลด VIP)
```

**ตัวอย่าง:** น้ำหนัก 2.5 กก., กรุงเทพฯ, บริการด่วน, VIP
1. ค่าจัดส่งพื้นฐาน = 60 บาท (ช่วง 1-3 กก.)
2. บวกค่าบริการด่วน = 60 + 30 = 90 บาท
3. หักส่วนลด VIP 20% = 90 × 0.80 = **72 บาท**
